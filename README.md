# 📊 Excel Data Visualization Project

## 📌 Project Overview

This project demonstrates how Microsoft Excel can be used to analyze and visualize survey data through a variety of statistical summaries and interactive charts. The project includes multiple data analysis tasks that transform raw data into meaningful insights using Excel functions, PivotTables, and different chart types.

The objective is to explore participant responses, identify trends and patterns, and present the findings using clear and visually appealing dashboards and charts.

---

## 🎯 Objectives

* Analyze survey data using Microsoft Excel.
* Summarize data using Excel functions and PivotTables.
* Visualize insights with appropriate charts.
* Understand participant demographics, investment behavior, and expectations.
* Present analytical findings in a professional and easy-to-understand format.

## 📂 Project Tasks

### Task 1 – Gender Distribution Analysis

**Objective**

* Analyze the gender distribution of participants.
* Calculate the count and percentage of each gender.
* Visualize the results using a Pie Chart.

**Visualizations**

* Pie Chart
* Summary Table
* Pivot Table

---

### Task 2 – Age Distribution Analysis

**Objective**

* Group participants based on age.
* Analyze the frequency of each age group.
* Visualize the distribution using a Histogram or Column Chart.

**Visualizations**

* Histogram
* Column Chart
* Pivot Table

---

### Task 3 – Investment Preferences Analysis

**Objective**

* Analyze the different investment avenues selected by participants.
* Compare their popularity.
* Present the results graphically.

**Visualizations**

* Bar Chart
* Column Chart
* Pivot Table

---

### Task 4 – Duration and Expectations Analysis

**Objective**

* Analyze investment durations mentioned by participants.
* Summarize participants' expectations from investments.
* Compare different investment expectations.

**Visualizations**

* Histogram (Investment Duration)
* Radar Chart (Investment Expectations)
* Pivot Tables
* Summary Tables
