# 📊 Gender Distribution Analysis in Excel

## 📌 Project Overview
This project demonstrates how to analyze the gender distribution in a dataset using Microsoft Excel. It uses Excel functions and PivotTables to calculate the number of records for each gender and visualizes the results with a pie chart.

## 🎯 Objective
- Analyze the gender distribution in the dataset.
- Calculate the count and percentage of each gender category.
- Create a visually appealing pie chart.
- Present the findings in an easy-to-understand format.

## 🛠️ Tools Used
- Microsoft Excel
- CSV Dataset
- COUNTIF Function
- Pivot Tables
- Pie Chart Visualization
